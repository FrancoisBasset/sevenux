# Sevenux

Sevenux is a fast-paced terminal card game inspired by the mechanics of Flip 7.

The project is built with Python and the **Textual** framework to explore modern terminal user interfaces (TUI) and experiment with card game mechanics in a lightweight CLI environment.

Sevenux aims to provide a clean, responsive and enjoyable card game experience directly in the terminal.

---

## Features

* Terminal UI powered by Textual
* Fast and lightweight gameplay
* Turn-based card mechanics
* Clean and modular Python architecture
* Designed for experimentation with TUI development

---

## Installation

Clone the repository:

```bash
git clone https://github.com/FrancoisBasset/sevenux.git
cd sevenux
```

Create a virtual environment (recommended):

```bash
python -m venv .venv
source .venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Run the Game

```bash
python sevenux.py
```

The game will launch directly inside your terminal.
